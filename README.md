# AI Pomodoro Timer

## Installation
#### NodeJS
Redwood requires Node version 18.
To switch versions:
  ```
  nvm install 18
  nvm use 18
  ```

#### Starting RW Dev Server
```
cd pomodoro
yarn install
yarn redwood dev
```
